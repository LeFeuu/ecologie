Comment réduire l’empreinte carbone d’un site web

  • Réduire le poids du site

  • Optimiser les images (compressées, format WebP/AVIF, tailles adaptées).
  • Charger les vidéos sur demande uniquement (lazy load).
  • Limiter l’usage des frameworks lourds si non nécessaires.
  • Réduire les JavaScript inutiles (éviter jQuery si pas utile, pur JS ou frameworks modernes optimisés).
  • Exemple de chargement paresseux | img src="image.webp" loading="lazy" alt="Description"|

  • Utiliser un hébergeur vert

  • Exemple : Infomaniak, o2switch, PlanetHoster, GreenGeeks, etc.
  • Vérifie via The Green Web Foundation.

  • Limiter les requêtes réseau

  • Fusionner les fichiers CSS/JS.
  • Utiliser la mise en cache ( ou en HTTP headers).
  • Minimiser les polices personnalisées (ex. Google Fonts coûteux en poids).
  • Charger une seule variante de police link href="#" rel="stylesheet"
